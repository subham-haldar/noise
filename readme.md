1. Clone this repo
2. Download **ffmpeg.exe** and **ffprobe.exe** from https://www.gyan.dev/ffmpeg/builds/ and paste into project folder
3. Download **yt-dlp.exe** from https://github.com/yt-dlp/yt-dlp/releases/latest and paste into project folder
4. Run `pip install flask flask-cors noisereduce scipy numpy`
5. Run `python app.py`
6. Open `http://127.0.0.1:5000`

## Tech Stack
HTML · CSS · JavaScript · Python · Flask · FFmpeg · yt-dlp · noisereduce
